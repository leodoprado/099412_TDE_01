# TDE Teoria da Computação - Mensurando Algoritmos de Ordenação (notação: Big O)

### Algoritmos:

- Bubble Sort (execução limitada 10³ para 10⁴)
- Merge Sort
- Quick Sort
- Counting Sort

### Entradas:

- 10³ = 1000
- 10⁶ = 1000000
- 10⁹ = 1000000000 (execução limitada)

## Getting started

### Install
```
$ pip3 install -r requirements.txt
```

### Run
```
$ python main.py
```
